# 猫毛様式吹き出しデザイン 技術設計書

## 目的

チャット画面の吹き出しデザインに「猫毛様式」(catFur)を追加する。吹き出しの外縁に沿って、不揃いな毛束（ストランド）を描画する装飾的なデザインを実現する。

## 形状の特徴

- 各ストランドは2本の二次ベジェ曲線（始点→頂点、頂点→終点）で構成
- 各曲線の膨らみ方向（外側/内側）はランダムに決定（前半・後半は同じ方向）
- 前半の曲線は70%の確率で外側に膨らむ
- 後半の終点はピーク位置から始点方向に戻った非対称な位置に配置
- 毛束の幅・高さが不揃い（手書き的な揺らぎ）
- 線の太さは均一（1.5pt）
- 四隅ではストランドの生え際の仮想線が角丸の円弧となり、その弧に沿って複数本のコーナーストランドを敷き詰めることで、辺と同じ毛並み密度で四隅も連続的に繋がる
- 各コーナーストランドのピークは、コーナー内側中心点から放射状に外側へ突き出す

## アーキテクチャ

既存のデザイン切り替え機能を拡張する形で実装する。

| レイヤー | コンポーネント | 変更内容 |
|---------|-------------|---------|
| Data | ChatBubbleDesign enum | `catFur` 値を追加 |
| UI | CatFurBubblePainter【新規】 | CustomPainter で毛束を描画 |
| UI | ChatBubbleDesignExtension | `catFur` ケースを追加 |
| Repository | ChatBubbleDesignRepository | 変更なし（enum.name で自動対応） |

## 主要コンポーネント

### 1. CatFurBubblePainter（CustomPainter）【新規作成】

**配置**: `client/lib/ui/component/cat_fur_bubble_painter.dart`

**コンストラクタ**:

```dart
CatFurBubblePainter({
  required Color backgroundColor,
  required int seed,  // 乱数シード（吹き出しごとに異なる形状を生成）
})
```

**描画処理の構成**:

1. **背景描画**: 角丸矩形（radius 12）を塗りつぶし
2. **ファーレイヤー描画**: 濃いグレー（`grey.shade600`, alpha 180）で
   - 各辺にストランドを配置（時計回りの毛流れ：上→下→左→右の順で描画し、下辺・左辺は描画方向を反転）
   - 各辺の最初/最後のストランドの端点を記録（`_EdgeEndpoints`）
   - 記録した端点を弧の始終点として、4つのコーナーに弧に沿った複数本のストランドを敷き詰める

**ストランド配置のアルゴリズム**:

各辺（上・下・左・右）に対して以下を繰り返す：

1. 角丸マージン（10pt）を避けた描画可能範囲を算出
2. 始点位置（`pos`）を `cornerMargin` から開始（下辺・左辺は逆方向のため `edgeLength - cornerMargin` から開始）
3. ストランドを描画し、戻り値の終点キャンバス座標から辺方向成分を取り出して次のストランドの始点 `pos` とする
4. 辺の終端まで繰り返し
5. 最初のストランドの始点・最後のストランドの終点（いずれもキャンバス座標）を `_EdgeEndpoints` として返却

**ストランド形状パラメータ（定数）**:

| パラメータ | 値 | 説明 |
|-----------|-----|------|
| `_minStrandWidth` | 5pt | 毛束の幅の最小値 |
| `_maxStrandWidth` | 16pt | 毛束の幅の最大値 |
| `_minPeakHeight` | 3pt | 毛束の高さの最小値 |
| `_maxPeakHeight` | 6pt | 毛束の高さの最大値 |
| `strokeWidth` | 1.5pt | 線の太さ（均一） |
| `_firstHalfOutwardBulgeProbability` | 0.7 | 前半曲線が外側に膨らむ確率 |

**ストランド幅の分布**: `1 - rand1 * rand2`（2つの独立した乱数の積を反転）により、最大値に近い幅が出やすい偏り分布を使用。

**1本のストランドの描画手順** (`_drawSingleFurStrand`):

1. 始点・頂点・終点の3点を辺座標系で算出
   - 始点: `position`（辺上、`_maxBaseOffset` の範囲で内側にずらす）
   - 頂点: `position + strandWidth`（外側に `peakHeight` 分突き出す）
   - 終点: 頂点位置から始点方向に `strandWidth * _endReturnRatio` の範囲でランダム配置（辺上、内側ずらしあり）
2. 始点→頂点、頂点→終点の2本の二次ベジェ曲線で弧を描く
   - 各曲線の制御点は弦の中点から辺の外側方向にオフセット
   - 膨らむ（正）か凹む（負）かはランダムに決定（前半・後半は同じ方向）
   - 膨らみ量: 1.5〜4.0pt
3. 弧の内側を背景色で塗りつぶし、下に重なる毛束の線を隠す
4. 弧の輪郭線を均一な太さ（1.5pt）で描画
5. 始点・終点のキャンバス座標 `({Offset start, Offset end})` を返す
   - 呼び出し側は終点の辺方向成分を次のストランドの始点として使う
   - また、辺単位での「最初のストランドの始点」「最後のストランドの終点」をコーナーストランドへ受け渡す用途にも使う

**コーナーストランドの描画手順** (`_drawCornerFurStrands`):

四隅の角丸マージン領域では、ストランドの生え際の仮想線を円弧として扱い、その弧に沿って複数本のストランドを敷き詰める。これにより、辺と同じ毛並み密度・テクスチャが四隅でも維持され、生え際が滑らかな角丸となる。

1. 引数として、前の辺の最後のストランドの終点 `edgeStart` と次の辺の最初のストランドの始点 `edgeEnd`（いずれもキャンバス座標）を受け取る
2. コーナー種別ごとに、内側中心点 `arcCenter`（バブル内部の `(±cornerMargin, ±cornerMargin)` 相当）を決定
3. 中心からみた `edgeStart`・`edgeEnd` の角度と距離を計算し、ストランドの生え際の仮想円弧を定義する（全コーナーで角度を増加させる方向に90度進む）
4. 円弧に沿ってストランドを敷き詰める:
   - 始点は `edgeStart` から開始し、以降は前のストランドの終点を次のストランドの始点として連結
   - 各ストランドの角度幅は `desiredArcLength / baseRadius` で算出（`desiredArcLength` は辺ストランドと同じ偏り分布）
   - ピークは弧上の点から放射状に外側へ `peakHeight` 分突き出す
   - 残り角度を上回りそうなストランドは最終ストランドとして扱い、終点を `edgeEnd` に合わせる
5. 各ストランドは2本の二次ベジェ曲線（始点→ピーク、ピーク→終点）で構成
   - 制御点は弦の中点から放射状方向（中心から弦中点方向）へオフセット
   - フィルは弧の内側を背景色で塗りつぶし、下に重なる線を隠す

コーナー間の接続関係（時計回り）:

| コーナー | 弧の始点 (`edgeStart`) | 弧の終点 (`edgeEnd`) |
|---------|------------------------|----------------------|
| 左上 | 左辺の最後のストランドの終点 | 上辺の最初のストランドの始点 |
| 右上 | 上辺の最後のストランドの終点 | 右辺の最初のストランドの始点 |
| 右下 | 右辺の最後のストランドの終点 | 下辺の最初のストランドの始点 |
| 左下 | 下辺の最後のストランドの終点 | 左辺の最初のストランドの始点 |

これにより、辺のストランドとコーナーストランドが同一キャンバス座標で接続され、生え際の仮想線が辺の直線部から角丸の弧へ滑らかに移行する。

**座標変換ヘルパー**:

- `_edgePoint`: 辺座標系（along, outward）を画面座標に変換
- `_bulgedControlPoint`: 2点を結ぶ弦の中点から辺の外側方向にオフセットした制御点を算出（辺用）
- `_radiallyBulgedControlPoint`: 中心点を基準に、2点を結ぶ弦の中点から放射状方向にオフセットした制御点を算出（コーナー用）

### 2. ChatBubbleDesignExtension の拡張

**変更内容**:

- `displayName`: `'猫毛様式'` を返す
- `shouldWithPointer`: `false` を返す（ツノなし）
- `buildBubble`: `CustomPaint` + `CatFurBubblePainter` で描画。`child.hashCode` を seed として使用し、吹き出しごとに異なる毛並みパターンを生成

## 既存デザインとの比較

| デザイン | 形状 | 実装方法 |
|---------|------|---------|
| 社内標準様式 | 角丸四角形（radius 8） | BoxDecoration + BorderRadius |
| 次世代様式 | 角丸四角形（radius 20/2） | BoxDecoration + BorderRadius |
| 調整済様式 | 7角形（角を削り取り） | CustomClipper + Path |
| 猫毛様式 | 角丸矩形＋毛束装飾 | CustomPainter |
